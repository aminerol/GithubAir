# GithubAir

## OverView
a small app that list the most starred Github repos that were created in the last 30 days

## Installation
just clone the repo
```bash
$ git clone https://github.com/aminerol/GithubAir.git
```
open the workspace file and run the app. voiiiiiila!!!!!!
